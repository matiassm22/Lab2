Sobre Nosotros

Somos Equipo Machin, un grupo de estudiantes de Ingeniería en Informática comprometidos con el desarrollo de soluciones tecnológicas modernas y eficientes. Nuestro enfoque principal es el desarrollo de aplicaciones web, utilizando herramientas actuales como HTML, CSS, JavaScript y tecnologías de contenedores como Docker.

Nuestras fortalezas

  • Responsabilidad y cumplimiento de objetivos
  • Adaptación a nuevas tecnologías
  • Resolución de problemas

Objetivo del proyecto

Este sitio web fue desarrollado como parte del Laboratorio 2, con el objetivo de aplicar conocimientos en desarrollo web y desplegar la aplicación utilizando Docker, permitiendo su ejecución en distintos entornos de forma consistente.
