Comandos básicos de Docker

1. Ejecutar un contenedor

Permite ejecutar una imagen de Docker en un contenedor.

docker run hello-world

2. Listar contenedores en ejecución

Muestra los contenedores que están activos.

docker ps

3. Listar todos los contenedores

Muestra todos los contenedores, incluso los detenidos.

docker ps -a

4. Construir una imagen

Construye una imagen a partir de un Dockerfile.

docker build -t nombre-imagen .

5. Detener un contenedor

Detiene un contenedor en ejecución usando su ID.

docker stop ID_CONTENEDOR

6. Eliminar un contenedor

Elimina un contenedor que ya no se utiliza.

docker rm ID_CONTENEDOR

7. Ver imágenes disponibles

Lista todas las imágenes descargadas en el sistema.

docker images

8. Eliminar una imagen

Elimina una imagen del sistema.

docker rmi nombre-imagen

9. Ejecutar en segundo plano

Permite ejecutar un contenedor en modo "detached".

docker run -d -p 8080:80 nombre-imagen

10. Ver logs de un contenedor

Muestra los registros de ejecución de un contenedor.

docker logs ID_CONTENEDOR

Conclusión

Estos comandos permiten administrar contenedores e imágenes en Docker, facilitando el desarrollo, pruebas y despliegue de aplicaciones.
