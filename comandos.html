Comandos Docker

Inicio Docker Comandos

Ejecutar contenedor

docker run hello-world

Ejecuta un contenedor de prueba para verificar que Docker funciona correctamente.

Contenedores activos

docker ps

Muestra los contenedores que están en ejecución.

Todos los contenedores

docker ps -a

Lista todos los contenedores, incluso los detenidos.

Construir imagen

docker build -t app .

Crea una imagen Docker desde un Dockerfile.

Ejecutar con puerto

docker run -d -p 8080:80 app

Ejecuta el contenedor en segundo plano y conecta el puerto al navegador.

Detener contenedor

docker stop ID

Detiene un contenedor en ejecución.

Eliminar contenedor

docker rm ID

Elimina un contenedor detenido.

Ver imágenes

docker images

Muestra todas las imágenes disponibles.

Eliminar imagen

docker rmi app

Elimina una imagen del sistema.
