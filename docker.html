Tutorial de instalación y uso de Docker

Guía paso a paso para instalar Docker en Windows y crear un contenedor web sencillo.

Descargar Docker Desktop

  1. Abrir navegador
  2. Ir a https://www.docker.com/get-started/
  3. Presionar “Download Docker Desktop”
  4. Descargar el instalador

Instalar Docker

  1. Ejecutar Docker Desktop Installer.exe
  2. Aceptar términos
  3. Usar opción WSL 2
  4. Instalar
  5. Reiniciar el equipo si lo pide

rimer arranque

  1. Abrir Docker Desktop
  2. Esperar estado “Running”
  3. Abrir PowerShell o Terminal
  4. Ejecutar:
docker run hello-world

Esto descargará la imagen y verificará que Docker funciona correctamente.

Crear proyecto

  1. Crear carpeta: C:\docker-lab-web
  2. Crear archivo index.html
  3. Agregar una página HTML básica

Crear Dockerfile

Crear archivo llamado Dockerfile (sin extensión)

FROM nginx:alpine
COPY . /usr/share/nginx/html

Construir imagen

En la carpeta del proyecto ejecutar:

docker build -t mi-web-html .

Esto crea una imagen basada en nginx.

Ejecutar contenedor

docker run -d -p 8080:80 --name mi-web-container mi-web-html
  • -d: ejecuta en segundo plano
  • -p 8080:80: conecta tu PC con el contenedor

Ver resultado

Abrir navegador:

http://localhost:8080

Se verá tu página funcionando.

Detener y eliminar

docker stop mi-web-container
docker rm mi-web-container
