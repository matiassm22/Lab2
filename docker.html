¿Qué es Docker?

Docker es una plataforma que permite crear, ejecutar y gestionar aplicaciones en contenedores, asegurando que funcionen igual en cualquier entorno.

Requisitos

  • Windows 10/11 64 bits
  • WSL 2
  • Virtualización activada
  • 4GB RAM mínimo

Instalación

  1. Descargar Docker Desktop
  2. Instalar el programa
  3. Reiniciar el equipo
  4. Abrir Docker Desktop

Verificación

docker run hello-world

Uso básico

docker build -t app .
docker run -d -p 8080:80 app
