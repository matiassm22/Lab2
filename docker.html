Instalación y uso de Docker

¿Qué es Docker?

Docker es una plataforma que permite crear, ejecutar y administrar aplicaciones dentro de contenedores. Los contenedores son entornos aislados que incluyen todo lo necesario para ejecutar una aplicación, como código, librerías y dependencias.

Requisitos del sistema

  • Windows 10/11 de 64 bits
  • Virtualización activada en la BIOS
  • WSL 2 instalado (Subsistema de Windows para Linux)
  • Al menos 4 GB de RAM

Pasos de instalación

1. Descargar Docker Desktop

Ingresar al sitio oficial de Docker y descargar Docker Desktop para Windows.

2. Instalar el programa

Ejecutar el instalador y seguir los pasos. Durante la instalación se configurará automáticamente WSL 2.

3. Reiniciar el equipo

Una vez finalizada la instalación, es recomendable reiniciar el computador para aplicar todos los cambios.

4. Abrir Docker Desktop

Ejecutar Docker Desktop y esperar hasta que indique "Docker is running".

5. Verificar instalación

Abrir una terminal (PowerShell o Git Bash) y ejecutar el siguiente comando: 

docker run hello-world

Si aparece un mensaje de bienvenida, significa que Docker está funcionando correctamente.

Uso básico de Docker

Construir una imagen

docker build -t nombre-imagen .

Ejecutar un contenedor

docker run -d -p 8080:80 nombre-imagen

Ver contenedores activos

docker ps

Detener un contenedor

docker stop ID_CONTENEDOR

Conclusión

Docker es una herramienta fundamental en el desarrollo moderno, ya que permite ejecutar aplicaciones de forma consistente en cualquier entorno, facilitando el despliegue y la portabilidad.
